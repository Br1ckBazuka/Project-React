# React.js Sample Project

This app is built using React.js + React Router v4 + Express.js

### Installation

Clone repository:
```sh
git clone git@github.com:edoko/react-js-sample.git
```
Before running:
```sh
npm install
```


## Run

### Local test

```sh
npm start
```

### Local+Server test

```sh
npm run test
```

### Server only test

```sh
npm run server
```

### Production build

```sh
npm run build
```
